Псевдокласи вибору одного елемента

Псевдоклас nth-child(n)

Даний псевдоклас дозволяє обрати лише один небхідний елемент у документі.
Параметром, який ми вказуємо, є число. Важливо пам'ятати, що підрахунок починається з одиниці (не з нуля).

Приклад

Якщо нам, наприклда, необхідно обрати другий елемент у списку, аби зробити його червоним, ми пишемо у таблиці стилів наступне:

li:nth-child(2) {
color: red;
}

Зручною може видатися властивість даного псевдокласу, якої він набуває, коли до значення у дужках додати "n" - тоді при такому написанні ми можемо обрати кожен другий (або інший, який нам необхідно) елемент у списку:

li:nth-child(2n)


Псевдоклас nth-last-child(n)

На відміну від попереднього псевдокласу, цей варіант дає можливість обирати необхідний елемент починаючи відлік від кінця списку (що буде дуже зручно у випадку дуже довгих списків). Запис аналогічний до попередньго, але додається слово "last".

Приклад

li:nth-last-child(2) {
color: red;
}

Другий елемент списку з кінця в даному прикладі стане червоного кольору.
