Scale

scaleX(x) Масштабування елемента по горизонталі (вісь X).

style="transform: scaleX(1.5)"

scaleY(y) Масштабування елемента по вертикалі (вісь Y).

style="transform: scaleY(0.5)"

scale(x,y) Масштабування елемента по горизонталі та по вертикалі. Значення менше 1 - зменшує елемент, більше 1 - збільшує елемент.

style=" transform: scale(-2, 1.5);"

scaleZ(z) Маштабування елемента по ocs Z.

style="transform: scaleZ(2);"

scale3d(x,y,z) Масштабування елемента тривимірному просторі.

style="transform: scaleZ(2, 1.2, 10); "
