transform: skewX ()

Наклоняет элемент на заданный угол по вертикали, то есть вдоль оси X. В скобках указываем необходимое значение угла.

Пример

В следующем примере элемент <div> наклоняется на 20 градусов по оси X:

<div>{
transform: skewX(20deg)
}
</div>

transform: skewY ()

Наклоняет элемент на заданный угол по горизонтали то есть вдоль оси Y. В скобках указываем необходимое значение угла.

Пример

В следующем примере элемент <div> наклоняется на 20 градусов по оси Y:

<div>{
transform: skewY(20deg)
}
</div>

transform: skew ();

Наклоняет элемент сразу по двум заданым параметрам: и по горизонтали, и по вертикали.

Пример

В примере элемент >div> наклоняется на 20 градусов по оси X и на 10 градусов по оси Y:

<div>{
transform: skew(20deg,10deg);
}
</div>
